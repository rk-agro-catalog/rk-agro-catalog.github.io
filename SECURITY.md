# Security Policy

## Supported Versions

This project is maintained with security updates for the latest version.

| Version | Supported          |
| ------- | ------------------ |
| Latest  | ✅ Supported       |
| Older   | ⚠️ Not Supported   |

## Reporting a Vulnerability

If you discover a security vulnerability in this repository, please report it responsibly:

1. **Do NOT** create a public GitHub issue
2. **Do NOT** post about it in public forums
3. **Email** the maintainers at: rk-agro-catalog@github.com

Please include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

We will:
- Acknowledge receipt within 48 hours
- Provide updates every 7 days
- Work on a fix promptly
- Credit you in the security advisory

## Security Best Practices

For users of this project:

- Keep your browser and plugins updated
- Review code before deployment
- Never commit sensitive data (keys, passwords)
- Use HTTPS when accessing deployed sites
- Report any suspicious behavior immediately

## Security Features

- MIT Licensed
- Public code review process
- Dependency management
- Branch protection on main branch

Thank you for helping keep RK Agro Catalog secure!
